# 프로젝트의 실행 방법
1. git clone https://github.com/jakesungjaekim/wanted-pre-onboarding-frontend.git
2. yarn install
3. yarn start

### 사용 가능한 경로
##### '/'
: 홈 , 빈 화면
##### '/signin'
: 로그인 화면
##### '/signup'
: 회원가입 화면
##### '/todo'
: 투두리스트 앱 화면
